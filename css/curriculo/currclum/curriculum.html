HTML CSS
Intermediário

HTML e CSS são pilares complementares na criação de páginas da web envolventes e bem estruturadas. O HTML oferece a base estrutural do conteúdo, definindo a hierarquia dos elementos, como títulos, parágrafos, imagens e links. Por sua vez, o CSS assume a responsabilidade de controlar a apresentação visual desses elementos, permitindo a definição de cores, fontes, margens e layouts, resultando em páginas esteticamente agradáveis.

Javascript
Avançado

JavaScript desempenha um papel crucial na criação de interatividade e dinamismo em páginas da web. Como linguagem de programação do lado do cliente, o JavaScript permite que os desenvolvedores criem experiências mais envolventes e interativas para os visitantes do site.
